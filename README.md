<h1>website using React.js & Json-server</h1>


### Installation

1. Clone the repo
   ```sh
   https://github.com/sabreenahmed21/Crud-App.git
   ```
2. install:
   ```sh
   npm install
   ```
3. Run the app:
   ```sh
   json-server --watch db.json --port 3004
   ```
   ```sh
   npm start
   ```
